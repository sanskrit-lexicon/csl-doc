=========================
 Mobile Friendly Display
=========================

.. |loc_link| raw:: html

   <a href="http://youtu.be/7j8kfhxPvdE" target="_youtube">YouTube tutorial</a>

The so-called *Mobile Friendly Display*  is functionally very similar to
the Basic Display, but it scales better to smaller display screens than does
the Basic Display.

Please see this |loc_link| for an
introduction to the features of the Mobile Friendly Display. 
This video tutorial 
assumes the viewer is already familiar with the :doc:`basic`.
